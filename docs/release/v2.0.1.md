# Vega 2.0.1

Vega 2.0.1 corrige o workflow de release dos RPMs para openSUSE Leap.

## Correções

- usa Rust e Cargo fornecidos pelo próprio openSUSE Leap;
- remove a etapa redundante de instalação via `rustup`, que dependia de
  `bash` e interrompia o workflow antes da compilação;
- restaura a publicação dos pacotes `vegad` e `lyra-vega-gtk` para openSUSE.

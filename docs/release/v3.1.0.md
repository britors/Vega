# Vega 3.1.0

Vega 3.1.0 adiciona configuração de tela ao módulo Tela e separa o monitor
de processos numa entrada própria de navegação.

## Melhorias

- adiciona a aba Monitor em Tela: resolução, taxa de atualização, escala e
  orientação, aplicadas via xrandr (X11) ou wlr-randr (Wayland/wlroots) na
  sessão gráfica ativa — sem exigir privilégio de sistema;
- move o monitor de processos (CPU, memória, disco, rede) para a entrada
  "Monitor do Sistema", separada de Tela na navegação.

## Empacotamento

- corrige o build Arch: desabilita LTO no PKGBUILD do `vega-gtk` (o LTO
  padrão do Arch quebrava o link do crate `ring`) e copia os arquivos
  `.install` dos pacotes para o diretório de build;
- corrige o build openSUSE: remove a action `dtolnay/rust-toolchain`, que
  quebrava o `PATH` do container Leap — o `zypper` já resolve Rust 1.96,
  acima do mínimo do projeto;
- corrige os workflows de release para publicar a release (antes ficavam
  presos como draft mesmo com os pacotes já anexados).

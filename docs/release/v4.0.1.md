# Vega 4.0.1

## Correção de release

A v4.0.0 tentou padronizar os três workflows RPM/deb com
`dtolnay/rust-toolchain@stable`, mas essa action falha dentro do container
`opensuse/leap:16.0` (`bash` não encontrado no `$PATH`), quebrando o build
do pacote openSUSE naquela release. O openSUSE volta a usar o rust/cargo
do próprio `zypper`, como sempre funcionou antes da 4.0.0.

## Ícone

Novo ícone do `vega-gtk`: fundo em degradê preenchendo todo o squircle,
com o glifo (a estrela de quatro pontas) em branco liso por cima, no lugar
do fundo escuro sólido com glifo colorido da versão anterior — visual mais
próximo do padrão de ícones nativos do GNOME (Editor de Texto, Contatos,
Clima).

Veja [v4.0.0](v4.0.0.md) para as demais mudanças desta versão (o
`vega-cli` e as correções de bugs do `vegad`).

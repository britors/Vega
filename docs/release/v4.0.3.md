# Vega 4.0.3

## Empacotamento

Adiciona a dependência de `secret-tool`/`libsecret` que faltava nos
pacotes `lyra-vega-gtk` do Arch (`libsecret`), Fedora (`libsecret`) e
openSUSE (`secret-tool`) — só o Debian já declarava isso
(`libsecret-tools`). O módulo Assistente de IA usa `secret-tool` para
guardar e ler credenciais no keyring do sistema; sem essa dependência
declarada, a falha só aparecia em tempo de uso, não na instalação.

Validado com `rpmbuild` real (Fedora e openSUSE) — confirmado via
`rpm -qp --requires` que a dependência aparece no pacote gerado. O
PKGBUILD do Arch não pôde ser validado com `makepkg` (indisponível no
ambiente onde esta mudança foi feita), só revisado por sintaxe.

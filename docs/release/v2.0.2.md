# Vega 2.0.2

Vega 2.0.2 melhora o gerenciamento de repositórios na interface de Software.

## Melhorias

- substitui o seletor por uma lista de repositórios;
- identifica visualmente cada repositório como ativo ou inativo;
- apresenta uma única ação contextual para ativar ou desativar a seleção;
- move a otimização de mirrors para o cabeçalho da página;
- atualiza o contrato D-Bus e os backends APT, DNF, Pacman e Zypper para
  informar o estado real dos repositórios;
- alinha a documentação de contribuição, migração e publicação no AUR à
  interface oficial Rust/GTK4.

# Vega 2.0.0

Vega 2.0.0 conclui a migração da interface para Rust, GTK4 e libadwaita. O
`vegad` permanece em Go e continua expondo operações privilegiadas por D-Bus
e polkit.

## Destaques

- interface GTK nativa promovida a implementação oficial;
- pacote e binário principal renomeados para `lyra-vega-gtk`;
- Electron, Node, npm e `chrome-sandbox` removidos dos pacotes finais;
- Painel, Software, snapshots, Backup, Hardware, Kernel, Armazenamento,
  Data/Hora, Rede/Firewall, Wi-Fi, Bluetooth, Usuários, Serviços, Logs,
  Assistente e Sobre disponíveis na nova interface;
- empacotamento atualizado para Arch, Fedora, openSUSE e Debian/Ubuntu;
- QA automatizado para Rust, Go, contratos D-Bus e política dos pacotes;
- mediana local de abertura de 215 ms, PSS de aproximadamente 64 MiB e um
  único processo de interface no ambiente de referência.

## Atualização

Os pacotes novos fornecem compatibilidade com o comando `/usr/bin/vega`. Em
caso de bloqueador, a tag `v1.8.1` permanece como ponto de rollback.

# Vega 4.0.5

## Empacotamento

Adiciona lançador de menu (`.desktop`) e ícone para o `vega-cli` — até
aqui só existia o lançador GTK (`vega.desktop`, aponta pro
`lyra-vega-gtk`); a interface de terminal (`dialog`) não tinha entrada
no menu de aplicativos e só era acessível abrindo um terminal e
digitando `vega`.

Novo `packaging/vega-cli/vega-cli.desktop` (`Terminal=true`, já que a UI
é `dialog` dentro do terminal) e `packaging/vega-cli/vega-cli.svg`
(janela de terminal com a estrela do Vega centralizada, reaproveitando o
gradiente da marca). Instalado pelos quatro empacotamentos (Arch,
Fedora, openSUSE, Debian) junto com o binário do `vega-cli`, e checado
no passo "Validar instalação" dos workflows de release.

# Vega 2.0.3

Vega 2.0.3 adiciona suporte nativo aos esquemas claro e escuro do GNOME e
refina a lista de repositórios.

## Melhorias

- acompanha automaticamente a preferência de tema do GNOME, inclusive quando
  ela muda com o aplicativo aberto;
- usa cores semânticas do libadwaita para fundos, cartões, textos e bordas;
- mantém os itens do menu legíveis no modo claro;
- coloca “Ativar” ou “Desativar” à direita de cada repositório;
- remove seleção e ícones redundantes da lista de repositórios.
